# Video Creator Tool
# AI短视频生产辅助工作台 Master Design

## 1. 项目背景

### 1.1 当前业务问题

当前跨境电商短视频生产流程：

产品
↓
人工策划
↓
人工编写剧本
↓
人工拆镜头
↓
判断拍摄方式
↓
真人拍摄 / AI生成
↓
剪辑发布

实际生产中存在以下问题：

#### 问题1：剧本到分镜转换效率低

人工剧本通常停留在内容描述层，无法直接指导摄影人员拍摄或AI视频模型生成，需要进一步拆解为结构化镜头。

#### 问题2：AI生成镜头准备成本高

AI视频生成效果依赖输入质量，需要明确：

- 首帧素材
- 产品参考图
- 动作参考视频
- 环境参考素材
- Prompt描述

#### 问题3：人工判断成本高

每个视频需要人工判断哪些镜头适合真人拍摄，哪些镜头适合AI生成，目前主要依赖个人经验。

## 2. 项目目标

建立一个：

剧本 → 分镜 → 生产决策 → AI视频生成

的半自动短视频生产工具。

## 3. 项目边界

### 本项目负责

- 剧本结构化解析
- 自动拆分视频分镜
- 分镜生产方式辅助判断
- AI镜头生产方案生成
- 素材需求管理
- Seedance等视频模型调用

### 本项目暂不负责

- 产品选品
- 市场趋势分析
- 爆款预测
- 用户需求挖掘
- 商业内容决策

以上属于未来商业内容决策系统。

## 4. 产品工作流

## 5. 核心模块设计

# Module 1：Script Analyzer

## 输入

- 产品信息
- 视频主题
- 人工剧本

## 输出

结构化镜头列表。

# Module 2：Storyboard Generator

负责：

剧本 → 分镜。

每个分镜包含：

- 镜头编号
- 时间范围
- 场景描述
- 镜头目标
- 动作描述
- 摄影语言
- 待确认生产方式

# Module 3：Production Decision

人工确认：

- 真人拍摄
- AI生成

系统保存每个镜头的生产方式。

# Module 4：Seedance Director

负责将AI镜头转换为模型可执行生产方案。

输出：

- 生成模式（T2V/I2V/V2V）
- 所需素材
- 首帧需求
- 参考视频需求
- Camera设计
- Motion设计
- Lighting设计
- Seedance Prompt
- Negative Prompt

# Module 5：Asset Manager

管理AI生成所需素材。

包括：

- 产品图片
- 多角度图片
- 首帧图片
- 环境参考图
- 动作参考视频
- 音频素材

状态：

- waiting
- uploaded
- ready

# Module 6：Video Generator

负责调用视频生成模型API。

输入：

- Prompt
- 图片
- 视频
- 音频

输出：

- 视频文件
- 任务状态
- 生成参数
- 成本记录

## 6. 人机协作原则

AI负责：

- 分析
- 拆解
- 生成建议
- Prompt转换

人工负责：

- 商业判断
- 分镜确认
- 最终审核

不追求完全自动化。

## 7. MVP版本规划

## V0.1：生产方案生成

目标：跑通剧本到AI生产方案。

功能：

- 剧本输入
- AI拆分镜
- 人工选择实拍/AI
- AI镜头Prompt生成
- 素材需求清单

暂不接视频API。

## V0.2：视频生成闭环

增加：

- 素材上传
- Seedance API调用
- 视频任务管理
- 结果保存

## V0.3：多模型支持

支持：

- Seedance
- Kling
- 即梦

## V1.0：连接商业决策系统

未来接入：

- 产品分析
- 爆款数据
- 用户反馈
- 内容效果反馈

## 8. 核心设计原则

### 原则1：AI辅助生产，不替代导演

### 原则2：先生成生产方案，再生成视频

### 原则3：所有过程可追踪

保存：

- 剧本
- 分镜
- Prompt
- 素材
- 生成参数
- 视频结果

## 9. 项目定位总结

Video Creator Tool不是一个商业决策系统，而是一个短视频AI生产执行工具。

短期目标：解决公司视频生产效率问题。

长期目标：积累视频生产数据，为未来商业内容决策系统提供数据基础。
